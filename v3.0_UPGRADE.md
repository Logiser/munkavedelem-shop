```bash

```

1. **Netlify Dashboard** → Automatikus build (Git push után)
2. Vár a build-re (~3-5 perc)
3. Deploy sikeres? ✅

## 📧 Email Beállítás (FONTOS!)

### 1️⃣ Gmail App Password

1. **myaccount.google.com** → Biztonság
2. **App passwords** → Jelszó generálása
3. **16 karakteres jelszót kapszol** (másolj!)

### 2️⃣ Netlify Environment Variables

1. **Dashboard** → **Settings** → **Build & deploy**
2. **Environment** → **Edit variables**

**Add:**
```
EMAIL_USER = your-email@gmail.com
EMAIL_PASSWORD = your-16-char-password
```

### 3️⃣ Test Order

1. Webshop → Válassz terméket
2. Kosárba → Checkout
3. Töltsd ki az adatokat
4. Kattints: **Rendelés Elküldése**
5. Ellenőrizd az email-jeidet 1-2 perc múlva

✅ Ha jön email az iroda@tuz-munkavedelmiszaki.hu-ra és az ügyfélnek is: **KÉSZ!**
